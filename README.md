# Simpletron-Simulator
* Programmer: Cong Trinh
* Class: CptS 122, Spring 2016; Lab Section 01
* Programming Assignment: PA7 Own Computer
* Date: April 6, 2016
* Description: This program simulates a simple microcontroller called the Simpletron. This program will interpret instructions (called Simpletron Machine Language (SML) instructions and carry out the corresponding operation.
* Function: To demonstrate the simpletron, the user is prompted to input memory[7] and memory[8]. The simpletron will carry out the operation of adding memory[7] and memory[8] and insert the value into memory[9].
